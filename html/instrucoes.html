📲 Instalação no Celular (Recomendado)

Como instalar no iPhone (Safari)

1
Abra o Safari e acesse o link do aplicativo
2
Toque no botão "Compartilhar" ⬆️ (quadrado com seta para cima) na barra inferior
3
Role para baixo e toque em "Adicionar à Tela de Início"
4
Toque em "Adicionar" no canto superior direito
✅ O aplicativo aparecerá na sua tela inicial com o ícone ⚡ e funcionará como um app nativo, sem a barra de endereço do Safari!

Como instalar no Android (Chrome)

1
Abra o Chrome e acesse o link do aplicativo
2
Toque no menu ⋮ (três pontos) no canto superior direito
3
Toque em "Adicionar à tela inicial" ou "Instalar app"
4
Confirme tocando em "Instalar" ou "Adicionar"
✅ O aplicativo aparecerá na sua tela inicial e funcionará como um app nativo!

📋 Preenchendo o Formulário

1. Identificação do Quadro

Preencha os dados básicos:

  • Nome do Quadro: Ex.: QD-01, QGBT, QE-Bombas
  • Local: Sala elétrica, Copa, área externa
  • Data: Preenchida automaticamente com a data atual
  • Responsável: Seu nome completo
  • Tensão: Selecione a tensão nominal do quadro
💡 Dica: Os dados são salvos automaticamente no navegador. Se fechar e reabrir, seus dados estarão lá!

2. Dispositivos Instalados

Marque os dispositivos presentes no quadro e informe a quantidade:

  • marque os itens presentes
  • informe a quantidade numérica
  • DPS, DR, contatores, relés, etc.

3. Checklist de Segurança

Para cada item, marque:

  • Sim: está em conformidade
  • Não: não está em conformidade
  • N/A: não se aplica
  • Obs: adicione observações relevantes

4. Circuitos (Lado A e Lado B)

Para cada circuito, preencha:

  • Descrição: Finalidade do circuito (iluminação, tomadas, etc.)
  • Proteção: Disjuntor (1P 16A, 2P 20A, etc.)
  • Fase: A, B, C ou combinações
  • Corrente: Medida com alicate
  • Estado: OK, Atenção, Inativo, Manutenção
💡 Use o botão "+ Adicionar linha" para mais circuitos

5. Resumo do Quadro

O sistema calcula automaticamente:

  • Total de circuitos
  • Circuitos por fase
  • Potência (baseado em corrente, tensão e FP)
  • Quantidade de DRs e DPSs
⚠️ Você pode editar manualmente a potência se o cálculo automático não coincidir com valores medidos.

6. Fotos

📷
Tirar Foto: Abre a câmera do celular para fotografar o quadro
📎
Anexar Arquivo: Seleciona fotos já existentes na galeria

Tire fotos do quadro aberto, das etiquetas, dos dispositivos, de qualquer irregularidade encontrada.

💾 Salvando e Exportando

Salvamento Automático

Todos os dados são salvos automaticamente no navegador do celular (localStorage).

✅ Se você acidentalmente fechar o aplicativo, os dados permanecem salvos.

Exportar Dados

Para enviar o formulário preenchido:

1
Toque no botão 📤 Exportar no topo
2
Um arquivo .json será baixado
3
Envie por e-mail, WhatsApp ou transferência

Importar Dados

Para continuar um formulário salvo:

1
Toque em 📥 Importar
2
Selecione o arquivo .json exportado anteriormente
3
Os dados serão carregados no formulário

Novo Formulário

Para iniciar um novo registro:

1
Toque em 📄 Novo Formulário
2
Confirme que deseja limpar os dados
⚠️ Isso apaga todos os dados do formulário atual. Exporte antes se precisar salvar!

🖨️ Imprimir / PDF

Para gerar um PDF ou imprimir:

1
Toque em 🖨 Imprimir / PDF
2
No iPhone: Role para baixo e toque em "Salvar como PDF"
3
No Android: Selecione "Salvar como PDF" como destino

⚠️ Problemas Comuns

Os dados desapareceram!

Se você limpou os dados do navegador ou usou modo anônimo, os dados podem ter sido perdidos. Sempre exporte o JSON ao final de cada vistoria.

A câmera não abre

Certifique-se de que o navegador tem permissão para acessar a câmera. No iPhone, use o Safari para melhor compatibilidade.

Quero continuar em outro dispositivo

Exporte o JSON, envie para si mesmo e importe no outro dispositivo.

1

2
